3. The Pipeline — Steps A to I

StepFunctionWhat it does
Awn.read_csv(path)Load CSV → time, values, meta
Bwn.plot_series(time, values, metadata=meta)Plot raw time series
Cwn.compute_msd(values)Compute empirical MSD → lags, msd
Dwn.plot_msd_empirical(lags, msd, metadata=meta)Inspect MSD shape
Ewn.fit_msd(lags, msd, model='...')Fit model → FitResult
Fprint(fit.summary())Print parameters + 95% CI
GmatplotlibOverlay fitted curve on MSD
HmatplotlibPlot displacement PDF
Iwn.analyze(path, model='...')Run A–H in one call
# Steps A–D
time, values, meta = wn.read_csv('mydata.csv')
wn.plot_series(time, values, metadata=meta)
lags, msd = wn.compute_msd(values)
wn.plot_msd_empirical(lags, msd, metadata=meta)

# Steps E–F
fit = wn.fit_msd(lags, msd, model='cosine')
print(fit.summary())

# Steps G–H (manual plots)
fig, ax = plt.subplots()
ax.scatter(lags, msd, s=12, color='#555', alpha=0.6, label='Empirical')
ax.plot(fit.lags_used, fit.msd_fitted, color='#1B3A6B', lw=2, label='Fitted')
ax.legend(); plt.show()

T = int(lags[len(lags)//10])
displacements = values[T:] - values[:-T]
mu_disp = np.mean(displacements)
model_info = wn.get_model('cosine')
sigma2 = model_info['msd'](T, fit.params['mu'], fit.params['nu']) * fit.params['N']
dx = np.linspace(displacements.min(), displacements.max(), 300)
pdf = np.exp(-(dx-mu_disp)**2/(2*sigma2)) / np.sqrt(2*np.pi*sigma2)
plt.hist(displacements, bins='auto', density=True, alpha=0.5, color='#888')
plt.plot(dx, pdf, color='#C0392B', lw=2); plt.show()

# Step I — equivalent one-liner
result = wn.analyze('mydata.csv', model='cosine')
result.summary()
wn.plot_diagnostics(result)

4. Memory Parameter μ

μRegime
< 0.95Subdiffusive
0.95–1.05Near-Brownian
1.05–2.0Superdiffusive
> 2.0Hyperballistic

For fBm: H < 0.5 subdiffusive  ·  H = 0.5 Brownian  ·  H > 0.5 superdiffusive

R² guide: ≥ 0.8 good  ·  0.5–0.8 moderate  ·  < 0.5 try another model

wn.print_comparison(cr)

7. Preprocessing (optional)

FunctionOptionsUse when
wn.detrend(values, method='linear')'linear', 'polynomial', 'mean', 'moving_average'Data has a trend
wn.normalize(values, method='zscore')'zscore', 'minmax', 'mean', 'maxabs'Data amplitude is very small or large
wn.smooth(values, window=5)'moving_average', 'gaussian'Noisy signal
fluct = wn.detrend(values, method='linear')
wn.plot_series(time, fluct, title='Detrended')

# Compare before/after
fig, axes = plt.subplots(2, 1, figsize=(10, 5), sharex=True)
wn.plot_series(time, values, metadata=meta, title='Original',  ax=axes[0], show=False)
wn.plot_series(time, fluct,  metadata=meta, title='Detrended', ax=axes[1], show=False)
plt.tight_layout(); plt.show()
Note: The N parameter in the fit (N · msd_model) absorbs data amplitude — μ is scale-independent. Normalize only if N collapses to ~0 in the fit output.
