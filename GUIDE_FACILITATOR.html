3. Complete Function Reference

I/O

wn.read_csv(path: str) → (time, values, meta)
  • Reads 2-column CSV with name [unit] headers
  • Returns: time (1D array), values (1D array), meta (dict)
  • Meta keys: x_name, x_unit, x_label, y_name, y_unit, y_label, source_file, n_points
  • Raises FileNotFoundError (✗ prefix) or ValueError for bad format
  • Prints ⚠ warnings V1–V4 (non-fatal)
wn.read_csv_multi(path: str) → list of (time, values, meta)
  • Multi-column CSV; column 1 = time, each other column = one dataset

Preprocessing

wn.detrend(values, method='linear', poly_order=1) → array
  • methods: 'linear', 'polynomial', 'mean'
wn.normalize(values, method='zscore') → array
  • methods: 'zscore', 'minmax', 'mean', 'maxabs'
wn.smooth(values, window=5, method='moving_average') → array
  • methods: 'moving_average', 'gaussian'

Core

wn.compute_msd(x, max_lag=None, normalize=False) → (lags, msd)
  • Default max_lag = N // 2 (hard-capped at N/2)
  • normalize=True divides all MSD values by MSD[lag=1]
wn.fit_msd(lags, msd, model='cosine', p0=None, bounds=None, max_lag_fraction=1.0) → FitResult | None
wn.list_models() → None # prints formatted table of all 16+ models
wn.get_model(name: str) → dict

Analysis

wn.analyze(source, model='cosine', label='', max_lag=None, max_lag_fraction=1.0, verbose=True) → AnalysisResult
  • source: CSV path string OR numpy array (array requires label=)
  • Returns AnalysisResult with: .summary(), .fit, .regime, .mu, .to_csv(), .values, .lags
wn.compare(datasets, model='cosine') → ComparisonResult
  • datasets: dict {label: path_or_array} or list of paths
  • Per-dataset models: model={'A': 'cosine', 'B': 'exponential'}
wn.batch_analyze(source, model='cosine', pattern='*.csv') → ComparisonResult
  • source: folder path string, list of paths, or multi-column CSV path

4. Models Reference

ModelParamsMSD formulaNotes
exponential μ, β, N Γ(μ)·β−μ·Tμ−1·e−β/T Most stable; good default starting point
cosine μ, ν, N Power-law × cos(νT/2) × Bessel Jμ−½ Oscillatory; valid range T < π/ν
sine μ, ν, N Same as cosine with sin(νT/2) Wider valid range T < 2π/ν; MSD=0 at T=0
fbm H, N T2H H=0.5 → Brownian; reports H not μ
dna a, b, c, N N·(a − c·e−bL) Restricted diffusion; no μ or regime
12 stubs — NotImplementedError Not yet implemented; use above 5
Facilitator tip: Start participants with 'exponential' — it converges reliably for most inter-event time datasets. Introduce 'cosine' only after they've seen a successful fit.

5. Memory Parameter μ — Full Regime Table

μ rangeRegimePlain-language description
μ < 0.95SUBDIFFUSIVEPast fluctuations resist change — slow spreading
0.95 – 1.05NEAR-BROWNIANWeak or no memory — near-ordinary random walk
1.05 – 2.0SUPERDIFFUSIVEPast fluctuations reinforce future ones — fast spreading
μ > 2.0HYPERBALLISTICExtreme memory — strongly persistent dynamics

For fbm model: H < 0.475 → subdiffusive; 0.475–0.525 → near-Brownian; H > 0.525 → superdiffusive.

7. Troubleshooting Guide

SymptomLikely causeFix
FileNotFoundError Wrong path or working directory Check current directory with import os; os.getcwd(); use relative path from notebook location
AttributeError: 'plot_msd_raw' Outdated notebook cell Use wn.plot_msd_empirical() (not plot_msd_raw)
TypeError: plot_series() got unexpected keyword 'xlabel' Old notebook cell Use metadata=meta instead of xlabel=
R² < 0.3 Wrong model for the data Try a different model; run wn.list_models()
N = 0.0000 in fit summary Data amplitude too small (optimizer drives N → 0) Normalize first: scaled = wn.normalize(values, 'zscore') then wn.analyze(scaled, ...)
PDF histogram much narrower than Gaussian Leptokurtic / intermittent data (bursty fluctuations) Physically meaningful — not a bug. Try a larger lag_index= value.
⚠ "Time column not strictly increasing" Data not sorted by time Sort by column 1 before saving the CSV
NotImplementedError for chosen model Stub model selected Use one of: cosine, exponential, sine, fbm, dna
Blank rows / NaN error from CSV Raw data export with empty lines pd.read_csv(f, skip_blank_lines=True).dropna(how='all').to_csv('clean.csv', index=False)
No plot appears in Colab Backend issue Add %matplotlib inline before imports; or import matplotlib; matplotlib.use('Agg')

8. DNA Model — Facilitator Notes

Key difference: The 'dna' model does not extract a memory parameter μ. It fits a restricted diffusion / plateau model: MSD saturates to a finite value at large lag. There is no regime classification (subdiffusive / superdiffusive) for this model.

MSD formula: MSD(L) = N · (a − c · e−bL)

ParameterMeaningPublished value
aPlateau MSD at large lag≈ 5.21
bRate at which plateau is reached≈ 0.0024
cExponential amplitude (must satisfy c < a)≈ 3.81
NNormalization scalar≈ 1.0
Facilitator tip: The plateau is only visible for large lags (L ≫ 1/b ≈ 400). The sample dataset has n=2000 points (max_lag=1000) specifically to make the plateau visible. If using a shorter dataset, the plateau may not appear and the fit may perform poorly.

9. CSV Format — Full Specification

RuleDetail
Header requiredRow 1 is always the header. No exceptions.
Header formatname [unit] — space before bracket optional
Column 1Always the independent variable (time, index, position…)
Column 2Always the observable (what was measured)
UnitlessUse empty brackets: flux []
Multi-columnSupported for batch: time + N observables → N separate analyses

Validation Warnings (V1–V4) — all non-fatal

  • V1: Time not strictly increasing → "⚠ Time column is not strictly increasing."
  • V2: Negative values in count/number columns → "⚠ Negative values found…"
  • V3: Time column values too large (possible column swap) → "⚠ Time column values look unusually large."
  • V4: Unrecognized time unit → "⚠ Unrecognized time unit '…'."
